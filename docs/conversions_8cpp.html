std::string infixToPrefix ( std::string  infixFormula)

Takes the infix formula given and converts into prefix formula.

Parameters
infixFormula

Reverses the given infix formula and iterates through the formula pushing "+" , "*", ">", "~", ")" into the stack while the remaining characters are added to prefix string variable until "(" is found. Then the stack is popped until ")" again adding the popped operators to the prefix string.

Returns
std::string prefixFormula
